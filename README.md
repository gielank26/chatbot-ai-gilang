# Chatbot AI Gilang

## Deskripsi
Chatbot ini merupakan aplikasi berbasis Artificial Intelligence (AI) yang dirancang
untuk menjawab pertanyaan pengguna secara otomatis menggunakan pemrosesan bahasa alami.

## Model AI
Chatbot ini menggunakan model AI berbasis Large Language Model (LLM) melalui layanan API
(OpenAI GPT).

## Peran AI
AI berperan dalam memahami bahasa alami pengguna, menjaga konteks percakapan,
serta menghasilkan respons yang relevan secara dinamis.

## Target Pengguna
- Pelajar dan mahasiswa
- Pengguna umum
- Pengembang pemula

## Teknologi
- Platform: GitHub
- Bahasa Pemrograman: Python (contoh)
- AI Model: OpenAI GPT

## Status Proyek
Proyek ini masih dalam tahap pengembangan awal.
